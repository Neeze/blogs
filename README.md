
# Nam Nam Blogs 

Nơi lưu trữ những bài viết, nghiên cứu và "bí kíp" thực dụng để kéo AI từ mây xuống đất. 

Viết bởi thằng sinh viên muốn đơn giản hóa mọi thứ phức tạp.

## 🚀 Setup để vọc hoặc đóng góp

Nếu bạn muốn viết bài hoặc sửa lỗi (cảm ơn nhé!), cứ làm theo các bước:

1. **Clone repo:**

```sh
git clone https://github.com/Neeze/blogs.git
```

2. **Cài hàng:**

```sh
pnpm install
```



3. **Lên lửa:**

```sh
pnpm run dev
```

## 📝 Quy tắc "hành nghề"

Toàn bộ nội dung nằm trong thư mục `src/content/docs`.

- Bạn có thể tạo topic mới hoặc viết tiếp vào topic cũ.
- Sử dụng cú pháp [Starlight](https://starlight.astro.build/) để trang nhìn cho "nghệ".
- Đừng ngại dùng [Mermaid](https://mermaid.js.org/) nếu muốn vẽ sơ đồ tư duy.
